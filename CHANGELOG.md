# Changelog

All notable changes to Zen AI will be documented here.

## [1.0.0] - 2025-07-18

### Added
- Initial release of Zen AI platform
- Multi-agent orchestration with LangChain & LangGraph
- React 19 frontend with Vite 8 and Tailwind CSS v4
- API Gateway with JWT-based authentication
- Firebase Google OAuth login
- Claude-style Artifacts panel with Monaco Editor
- Razorpay subscription and billing system
- Qdrant vector database for RAG
- Tavily web search integration
- AWS S3 file storage support
- PDF and PowerPoint generation agents
- Redis session caching
- Docker support for all services
